About me

--

Hi, my name is Vasiliy Triandafilidi. I am a 2nd year Ms student at University of British Columbia working in Prof. Savvas Hatzikiriakos' lab [lab]

In my research I focus on investigating properties of matter using Method of Molecular Dynamics [md] Unlike many other methods of simulation, where people try to find how can they make equations more difficult so we have an equation that can predict more stuff, Molecular Dynamics focuses on one equation: Newtons equations of motion.

By solving numerically Newtons equations of motion for each atom explicitely we obtain a trajectory of every atom in the system, for every timestep. This alone can provide a visual insight on the world that could not be seen by any tool explicitely. But method of MD allows us to calculate a thermodynamically averaged parameters, such as Temperature, Pressure, Density ...

Tools:

To do Molecular Dynamics simulation I use open source code [lammps] , to analyze trajectories I use a powerful Python based package [mdanalysis] , to vizualize my results I use [vmd] .

So why do I write this blog?

--

I write this blog mostly because I want to keep track of the stuff that I am doing, as well as maybe share something useful with people and hear some feedback. I spend a lot of time working on my project and I hope the stuff that I am doing meanwhile maybe useful for other people as well. Molecular Dynamics trajectories require a lot of post processing work, that needs to be done. I do it using a powerful a tool : Python. I am a big Pythonista, i use it everywhere: I write this blog using Python based tool [pelican], I perform analysis using Python based package [mdanalysis] , I plot graphs using Python.
